# **Architecture Review Board (ARB)**

This directory contains all documentation related to the Architecture Review Board (ARB). The ARB is responsible for reviewing and approving major architectural decisions to ensure alignment across teams and domains.

### **Purpose of the ARB**
- Review and approve ADRs.
- Ensure cross-domain alignment.
- Provide guidance on architectural decisions.

2. **Submit ADRs**: Follow the [ADR Template](../adr/template.md) to propose architectural decisions.
